# Archived Superpowers Artifacts

This directory keeps historical workflow artifacts that were generated during
earlier planning and implementation phases. They are preserved for provenance
only and are not the current runtime, adapter, account-model, or testing
specification.

Use the public docs as the current source of truth:

- `README.md`
- `docs/adapter-capabilities.md`
- `docs/testing-strategy.md`
- `docs/developer_guide/spec_exec_testing.md`

OMX planning evidence for the cleanup itself remains under `.omx/specs`,
`.omx/interviews`, `.omx/context`, `.omx/plans`, and `.omx/ultragoal`.
